# Navigation

Ce dossier contient la configuration de navigation de l'application.

## Structure recommandée

- `index.tsx` - Configuration principale de la navigation
- Types de navigation personnalisés si nécessaire

## Note

Pour utiliser React Navigation, installez les dépendances nécessaires :

```bash
npm install @react-navigation/native @react-navigation/native-stack
npx expo install react-native-screens react-native-safe-area-context
```
